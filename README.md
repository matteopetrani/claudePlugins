# Claude Plugins

A collection of custom plugins for Claude Code.

## Overview

This repository contains a marketplace of Claude plugins created by Matt.

## Available Plugins

### Market Research and Idea Validation
A plugin for conducting market research and validating business ideas.

- **Location**: `./plugins/market-research-idea-validation`
- **Version**: 1.0.0
- **License**: MIT

## Structure

- `.claude-plugin/` - Marketplace configuration
- `plugins/` - Individual plugin implementations

## Installation

To use these plugins with Claude Code, add this repository as a marketplace.
